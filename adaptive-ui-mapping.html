How to Read This
Read by row →
For this reason, how does it manifest?
Physiological needs are handled quietly in Zone 1 only.

Mental model spans all three zones — from invisible terminology adjustments to conscious layout suggestions.

Behavioral changes grow in depth as data accumulates — from subtle Zone 1 tuning to Zone 3 proposals.
Read by column ↓
In this zone, what drives the change?
Zone 1 is driven by physiology and early signals — available from day one.

Zone 2 is driven by mental model and behavioral patterns — needs some data.

Zone 3 is driven by mental model and behavioral depth — needs strong, consistent signals before suggesting.

Zone 4 is available from day one but its value compounds with profile depth — the options surfaced narrow and sharpen as the system learns what fits this user.
Note on Zone 4 →
Same change-set as Zones 1–3, different path.
Zone 4 doesn't invent new changes. It gives the user a control surface over the same elements the system already adapts — with one twist: the options offered are scoped by the user's profile, not a universal settings menu. A power user sees "add bulk actions to toolbar" as a choice; a casual user doesn't. That's what separates it from a generic preferences page.
